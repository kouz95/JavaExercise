# 기능 구현 목록
- 카드 클래스 생성
    - 총 52장이다.
    - 게임 마다 무작위로 섞인다.
    - 카드는 Pair 데이터 형태로 저장.
- 게이머
    - 게이머는 Draw를 할 수 있다.
    - 게이머는 게임 결과를 호출 할 수 있다. (종료 선언)
    
- 딜러
    - Draw 기능 공유
    - 점수가 16점 이하면 한장 더 뽑을 수 있다.
    
- 점수 계산 기능
    - A = 1
    - K,Q,J = 10
    
- Draw 기능

- 딜러 상태 세팅 기능
    - 게임 한판 당 생성 즉시 결정됨.
    
- 점수 비교 및 승리 판단 기능
